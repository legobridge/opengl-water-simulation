void framebuffer_size_callback(GLFWwindow* window, int width, int height)

Callback function for Framebuffer resizing

void key_callback(GLFWwindow* window, int key, int scancode, int action, int mods)

Callback function for keypress processing-

  • Esc key closes the window
  • W key moves camera forward
  • S key moves camera backward
  • A key makes camera strafe left
  • D key makes camera strafe right
  • P key toggles the day/night cycle
  • Left Arrow key slows down the day/night cycle
  • Right Arrow key speeds up the day/night cycle

void cursor_pos_callback(GLFWwindow* window, double xpos, double ypos)

Callback function for mouse movement processing. Moving the mouse pans the screen in the direction of movement

void scroll_callback(GLFWwindow* window, double xoffset, double yoffset)

Callback funtion for mouse scrolling processing. Scrolling up increases camera speed, and scrolling down decreases camera speed
